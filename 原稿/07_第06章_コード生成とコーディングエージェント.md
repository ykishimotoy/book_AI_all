## 6-1　まず、2026年上半期に何が起きたか

この分野を語るには、勢力図の前に「事件」から入るのが正しい。わずか数ヶ月で、開発者の足元は3度揺れました。

### 事件①　Gemini CLI終了──「無料の梯子」は外される

Googleの**Gemini CLI**は、無料で1日1,000リクエストという業界最強の入口として、個人開発者に広く愛されたツールでした。それが2026年5月19日に終了が発表され、**6月18日に個人向け提供が停止**。約1年での退場です。後継の**Antigravity CLI（agy）**はGo製・クローズドソースで、クレジット従量課金制。オープンソースだった旧CLIとは思想ごと別物になりました（企業ライセンス・有料APIキーでの利用は継続）。

### 事件②　Copilotの課金大転換──定額制の終わり

GitHub Copilotは2026年4月20〜21日、個人プランの新規受付を突然停止し、既存ユーザーの利用制限を厳格化しました。その後の展開は速く、4月27日に従量課金化を正式発表、5月12日（米国時間）に新しい上位プラン**Max**を発表、そして**6月1日から全プランで「GitHub AIクレジット」従量課金が稼働**しました。コード補完と次編集提案は無料のまま、チャット・エージェント・コードレビューがクレジットを消費し、超過は1クレジット=$0.01。新規登録は6月17日（米国時間）に再開が発表され、以降段階的に再開されています。

### 事件③　GPT-5.6 ultraモード──性能競争の新局面

OpenAIが6月26日に発表したGPT-5.6のフラッグシップSolは、サブエージェント並列の「ultraモード」でTerminal-Bench 2.1の91.9%という新記録を主張。エージェント性能の競争は「1体の賢さ」から「並列チームの運用」へ移りつつあります（GA待ち・2026年7月時点）。

3つの事件が教える教訓はひとつです。**この分野では、ツールの優劣より「変化に耐える使い方」のほうが重要である。**具体策は本章の後半で扱います。

## 6-2　勢力図：CLI型・IDE型・自律型・OSS

**表6-1　コーディングAI勢力図（2026年7月時点）**

| 型 | 代表 | 特徴 |
|---|---|---|
| CLI型（ターミナル常駐） | **Claude Code**・Codex CLI・Antigravity CLI・Copilot CLI・Kiro CLI | エージェントとしてファイル操作・テスト実行まで自律実行。2026年の主戦場 |
| IDE型（エディタ統合） | Cursor・Windsurf・GitHub Copilot（IDE内） | 補完とチャットが中心。学習コストが低い |
| 自律型（委任特化） | Devin | タスクを丸ごと委任。PM的に使う |
| OSS | Cline・Aider・OpenCode | 無料・改造自由・モデルは自前調達 |

### CLI型の主要プレイヤー

**Claude Code**（Anthropic）は個人開発者の定番の座にあります。サブスクはPro $20から、ヘビーユーザー向けにMax 5x $100／Max 20x $200、チーム向けTeam Premiumが$100/席。SWE-bench Verifiedで85〜89%級（Sonnet 5／Opus 4.8、2026年7月時点）の性能に加え、並列サブエージェント（Agent Teams）・Skills・MCP対応と、機能面の先行が特徴です（6月30日リリースのSonnet 5が全プランで利用可能）。注意点は並列実行のコストで、サブエージェント3体の並列は単独実行の約3〜4倍のトークンを消費します。

**Codex CLI**（OpenAI）はApache 2.0のオープンソースであることが最大の差別化です。ChatGPTのサブスク（Go $8〜／Plus $20／Pro $100・$200）で利用でき、GPT-5.6のGA後はSol/ultraモードが載る見込みです（2026年7月時点）。そして見逃せないのが**無料枠の存在**です。ChatGPTアカウントがあれば追加契約なしに制限付きで試せるため、Gemini CLI亡きあと、**「コーディングエージェントを初めて触るビギナーの入口」としてはCodexの無料枠が現在の第一候補**です。まず無料で「エージェントに任せる感覚」を体験し、足りなくなったら課金の判断をする──この順番が、2026年のこの分野への正しい入り方です。

**Antigravity CLI**（Google）は、デスクトップ・CLI・SDK・IDEの4面で同一エージェントハーネスを提供する野心的な構成。料金はAIクレジット$0.01/個、$199で2万クレジット。ただし無料枠のクォータの渋さ（「数リクエストで週次上限」との報告）は7月時点でも大きな改善が報じられていません。

**GitHub Copilot**は、OpenAI・Anthropic・Google・xAIのモデルを切り替えられる**マルチモデル**が独自価値です。プランはFree／Pro $10／Pro+ $39／Max（6月新設）／Business $19／Enterprise $39。CLI版もGA済みで、Plan Mode・Autopilot・MCPに対応しています。

**Kiro CLI**（AWS）はIAM統合と知財補償を武器にしたエンタープライズ向けです。

### IDE型・自律型・OSS

IDE型では**Cursor**（$20）と**Windsurf**が定番。ターミナルに抵抗がある人の入口として依然有力です。自律型の**Devin**はCore $20＋作業量に応じたACU課金（$2.25/ACU）で、実用的には月$100〜300を見込む世界です。OSS勢は**Cline**（GitHubスター約64k）、**Aider**（約47k）、**OpenCode**が代表格で、モデルAPIを自前契約して組み合わせます。

## 6-3　料金の目安

- **無料で始める**：Codex CLIの無料枠（ビギナーの第一候補）、Copilot Free、OSS＋ローカルLLM（第5章）
- **個人の標準**：月$10〜20（Copilot Pro $10、Claude Code Pro $20、Cursor $20）
- **ヘビーユース**：月$100〜200（Claude Code Max、ChatGPT Pro、Copilot Max）＋従量課金分。エージェント並列を多用するなら、この帯を覚悟する

2026年の実務感覚として重要なのは、**「サブスク価格」より「クレジット消費速度」**です。同じ$20でも、補完中心なら余裕、エージェント委任中心なら数日で上限、ということが普通に起きます。最初の2週間は消費ペースの観測期間と考えてください。

## 6-4　日本語対応

主要ツールはいずれも日本語での指示・応答に対応しており、日本語だけで開発を進められます。差が出るのはドキュメントとコミュニティです。この点、日本語の解説記事・書籍が最も厚いのはClaude CodeとCopilotで、トラブル時に日本語情報へたどり着きやすいことは、初中級者には実質的な性能差として効いてきます。

## 6-5　注意点──「梯子を外されない」ための4原則

1. **設定資産をポータブルに保つ。**Claude CodeのCLAUDE.md、各種エージェントのAGENTS.mdのような「プロジェクトの説明書」は、書き方がほぼ共通です。この設定ファイル群こそが乗り換え時の保険。特定ツールの独自機能に深く依存する前に、汎用的な形式で知見を蓄積する
2. **無料枠を前提に業務を設計しない。**Gemini CLIの教訓。無料の入口は「試用」であって「基盤」ではない
3. **従量課金の上限を設定する。**エージェントは失敗時もトークンを消費する。暴走時の上限（ハードリミット）を必ず設定
4. **モデルとツールを分けて考える。**ツール（Copilot等）はマルチモデル化が進んでいる。「ツールへの不満」と「モデルへの不満」を切り分けると、乗り換え判断が正確になる

## 6-7　この章のまとめ

- 2026年上半期の3大事件（**Gemini CLI終了・Copilot従量課金化・GPT-5.6 ultra**）が示すのは、「ツールの優劣」より「変化に耐える使い方」の重要性

> **コラム：エージェントに「任せる」とは、レビューが仕事になること**
>
> コーディングエージェントを使い始めた人が最初に驚くのは、コードを書く時間が減る代わりに、コードを読む時間が増えることです。エージェントは大量のコードを高速に生成しますが、その品質保証はあなたの仕事として残ります。「書く人」から「レビューする人・方針を決める人」への役割転換──これはこの後の第7章（作業エージェント）でも、まったく同じ構図で現れます。AIに委任する時代のスキルとは、実は「指示力」と「検収力」なのです。
